## 生成矩阵

* 直接法
```m
a = [1, 2, 3; 4, 5, 6; 7, 8, 9];
```

* 冒号一维矩阵 a = 开始：步长：结束，步长为1可以省略
```m
b = 1:1:10 %1,2,3,4,5,6,7,8,9
```

* 函数生成
```m
    % linspace(开始，结束，元素个数)，等差生成指定元素的一维矩阵，省略则生成100个
c = linspace(0, 10, 5);
    
    %特殊矩阵
    e = eye(4); %eye(维数)单位阵
    z = zero(1,4); % zeros(维数)全零阵
    o = ones(4,1); % ones(维数)全1阵
    r = rand(4); %rand(维数)0~1随机分布阵
    rn = randn(4); %randn(维数)0
```
